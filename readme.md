## 搭建步骤
1. 下载安装c51v960a(或更高版本)，安装相应插件
2. 打开keil创建工程模板，方便后工程直接复制模板在模板上更改  
具体为:   
project-new uvision project 后保存  
选legacy device database-at89c51确认  
flash-configure flash tool-output-create hex file勾上  
后在工程下添加main.c
